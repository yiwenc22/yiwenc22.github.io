## Why I Like Software Engineering
  The world is changing fast and everything going be robotic later on. Software Engineering going to teach me a lot of things before that time come. Software Engineering is like one of the highest wanted positions in the field of computer science, so learning this skill going to help find a job much easier. Some other reason, I want to learn about software Engineering is that I want to develop a game and websites. Software engineering going to teach me all the basics to create a website and games. I am also interested in machine learning and data science, I always want to know how do machine learning work and what is data science. And my problem-solving skills going up after learning software engineering because of all the bugs or problems I may face when developing a website.
  
## Skill and Experiences After Software Engineering
  Some skill I want to learn or master after Software Engineering is creating a website from scratch without any template or helps from others. Able to create games on chrome that are playable. I also excepted my writing skills and question-asking skill to improve. So, in the future, I can write better and ask smart questions. Some other skills I will expect to have after are Object-oriented design, logical thinking, teamwork, and Debugging. Some experiences I hope are a chance to learn about machine learning, create games with HTML, software development, create at least 3 websites, and a lot of debugging chance. 
  
## Skill and Experiences In The Future
  Some skills I hope to have in the future are master in java, C language, HTML, javascript, and Python. Other skills can include complex problem solving, game development, Unity, etc. Some experiences I want for the future create a well-known game in Unity or chrome, an internship in the field before graduation, meet friends in the field, and have good knowledge of database science.
