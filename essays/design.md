## Software Engineering Reusable Solution
In Software Engineering, there are somethings called the design patterns. According to Christopher Alexandr design pattern describes a problem that occurs over and over again in our environment, and then describes the core of the solution to that problem, in such a way that you can use this solution a million time over, without ever doing it the same way twice. I would summary this as pattern help save brain cells.
