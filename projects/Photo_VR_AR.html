Combination of Project build with Unity

Project 1: AR Dodging Game

The first project is an AR-based dodging game, where a character is spawned on a detected playing card. The main objective is to dodge incoming objects, testing the player's reflexes and quick thinking.

  • AR and XR+ software used for card pattern detection and character display
  • Unity engine for game development

Project 2: VR Gladiator Game

This VR project takes players into a thrilling gladiator arena, where they must survive wave after wave of enemies. Players can grab weapons and use them to fight off adversaries while collecting potions and buffs to stay alive.

  • Developed using Unity
  • Utilizes the XR package for rigging and grabbing mechanics

Project 3: AR Tower Defense with Photogrammetry

The final project combines AR and photogrammetry to create an immersive tower defense game. Players can place the game board in a real-life environment, and the enemies are created using LIDAR-scanned 3D models generated by photogrammetry.

  • Developed with Unity
  • Meshroom software used to generate 3D models from photogrammetry
  • Real-life environments and objects integrated using LIDAR scanning

These three projects showcase the power and versatility of Unity, VR, AR, and photogrammetry technologies, demonstrating the potential for creating innovative and engaging experiences in gaming and beyond.
