LamTech-ReportSight

Overview

ReportSight is an intelligent reporting application developed for the Hawaii Annual Code Challenge (HACC) to assist Hawaii Marine Animal Response (HMAR). The platform enables HMAR workers to manage report forms with precise location data and comprehensive information.

Built with Meteor, React, and Bootstrap, ReportSight features a user-friendly interface that allows reporters to submit marine animal sighting forms efficiently. The application leverages AI-enhanced location services and data processing to improve reporting accuracy and response times.
