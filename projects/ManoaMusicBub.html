ManoaMusicBubs

Final Project - ICS 314

Overview

ManoaMusicBubs is an intelligent networking platform that connects University of Hawaii students with musical talents. The application uses AI-powered matching algorithms to help students find others with similar musical tastes and compatible abilities.

Students can create detailed profiles showcasing their musical preferences, capabilities, and goals—ranging from casual jam sessions to forming performing bands. Profiles include links to YouTube videos or SoundCloud tracks demonstrating their musicianship, enabling authentic connections through shared musical interests.

My Contributions

As a core programmer, I was responsible for:

  • Profile Management: Developed the profile page system allowing users to update information after account creation
  • Authentication: Implemented sign-up and sign-in pages with secure credential management
  • Database Design: Designed and implemented the default database schema for the application
  • User Data: Created systems for managing user email and password data with security best practices

What I Learned

This project reinforced the power of collaboration—working with a team proved invaluable when facing challenges. We supported each other through debugging sessions and learned alternative problem-solving approaches.

The debugging process was particularly educational. The site didn't always function as initially planned, but through persistence and teamwork, we systematically resolved issues and improved both the site's appearance and functionality.
