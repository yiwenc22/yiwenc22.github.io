Toyota Hawaii Care Claim Automation

Overview

During the COVID pandemic, Servco faced staffing shortages that slowed claim validation processes and led to errors. This project automated the Toyota Hawaii Care claim validation system using AI-powered decision logic and intelligent database processing.

The system uses API calls to execute SQL stored procedures that perform comprehensive validation checks. Claims are automatically processed if they pass all validation criteria, significantly reducing processing time and eliminating human error.

My Contributions

As one of two software engineers on the project, I was responsible for:

  • Database Logic: Designed and implemented all SQL stored procedures for claim validation
  • Validation Rules: Created complex validation logic based on business requirements from claim managers
  • Example Implementation: Developed validation checks for claims requiring cars purchased in Hawaii within 2 years, including location verification and date comparison logic
  • API Integration: Ensured seamless integration between API endpoints and database stored procedures

The system validates multiple criteria simultaneously, checking purchase location, purchase date, vehicle eligibility, and other business rules before processing claims automatically.
