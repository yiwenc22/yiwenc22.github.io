Covid-Tracker

Overview

Covid-Tracker is an intelligent self-screening platform that enables individuals to provide daily health check-ins before leaving home, entering work environments, or accessing campuses. The platform features AI-enhanced symptom analysis and data tracking.

Key features include daily symptom check-ins, comprehensive health history tracking, and secure vaccine card management with upload and editing capabilities. This project served as a skill refresh for Meteor development and preparation for the Meteor Hackathon.

What I Learned

This project helped me refresh my Meteor and Semantic UI skills. I learned modern deployment practices by researching and implementing online deployment strategies.

I significantly improved my understanding of React's image handling, discovering React-specific approaches after traditional HTML methods proved insufficient. Additionally, this collaborative project taught me effective teamwork, using Google Project Board for task management, and learning alternative problem-solving approaches through team collaboration.

My Contributions

I focused on frontend development across multiple pages:

  • Landing Page: Designed layout and simplified user experience
  • Home Page: Developed navigation bar and implemented CSS styling for text and icons
  • Profile Page: Collaborated with the team to implement vaccine card upload and display functionality

Key challenges included implementing image display after upload and finding alternative methods to retrieve username data beyond Meteor's default implementation.
